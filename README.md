# Ouchi Colon ころころレース v2

スーパーファミコンのマリオカート風・疑似3Dブラウザレースゲームです。  
GitHub Pages で公開できる **index.html 単体**で動作します。

---

## 遊び方

### スタート画面
- **スタート** ボタンを押す → キャラクター選択へ
- 下部の「キャラクター」ボタンでもキャラクター選択へ進めます

### キャラクター選択画面
- 中央のカードで選択中キャラの性能（そくど・ハンドリング・おもさ）を確認
- 下のグリッドでキャラを選択（現在は MIKARO のみ選択可）
- **けってい** ボタンでプレイへ進む
- **← もどる** でスタート画面へ戻る

### プレイ画面
| 操作 | PC | スマホ |
|---|---|---|
| 左移動 | ← キー | 画面左側タップ |
| 右移動 | → キー | 画面右側タップ |
| 加速   | ↑ キー | 中央 ▲ ボタン |
| 減速   | ↓ キー | — |

#### 画面の見方
- 左上：**LAP 1/3**（現在のラップ）
- 右上：**★ × 05**（集めた星の数）
- 右下：速度メーター（km/h）
- 左下：ミニマップ

## 開発者向け

### 画像素材パス一覧

以下のパスに PNG を置くと自動的に優先表示されます。  
ファイルが存在しない場合は Canvas 図形によるフォールバックが使われます。

```
assets/
  ui/
    title_logo.png          ← タイトルロゴ（透過PNG）
  backgrounds/
    living_room_bg.png      ← リビング背景（スタート画面・プレイ画面共用）
  characters/
    mikaro_front.png        ← MIKARO 正面（選択画面・スタート画面）
    mikaro_back.png         ← MIKARO 背面（プレイ画面プレイヤー）
    pitokoro_front.png
    marinkaba_front.png     ← マリンカバ
    fuwaro_front.png
    moolo_front.png         ← ムーロ / 敵キャラ Moolo
    kakubake_front.png
    takoron_front.png
    zono_front.png
  items/
    star.png                ← 星アイテム（透過PNG）
    block_blue.png          ← ブロック障害物（透過PNG）
```

画像サイズの目安：  
- キャラ画像: **256×256px** 以上の正方形（透過PNG推奨）  
- 背景画像: **390×844px** 程度（縦長、スマホ画面比率）  
- アイテム画像: **128×128px** 以上（透過PNG）

### ファイル構成

```
index.html          ゲーム本体（1ファイル完結）
README.md           本ファイル
docs/
  PROJECT_JOURNAL.md  開発記録
assets/
  reference/        デザイン参考画像
```

### GitHub Pages への公開

1. リポジトリを GitHub へ push
2. Settings → Pages → Source: `main` ブランチ `/` (root) を選択
3. 公開された URL で遊べます
