# Ouchi Colon ころころレース v2

スーパーファミコンのマリオカート風・疑似3Dブラウザレースゲームです。  
GitHub Pages で公開できる **index.html 単体**で動作します。

---

## 遊び方

### スタート画面
- **スタート** ボタンを押す → キャラクター選択へ
- 下部の「キャラクター」ボタンでもキャラクター選択へ進めます

### キャラクター選択画面
- 中央のカードで選択中キャラの性能（そくど・ハンドリング・おもさ）を確認
- 下のグリッドでキャラを選択（現在は MIKARO のみ選択可）
- **けってい** ボタンでプレイへ進む
- **← もどる** でスタート画面へ戻る

### プレイ画面
| 操作 | PC | スマホ |
|---|---|---|
| 左移動 | ← キー | 画面左側タップ |
| 右移動 | → キー | 画面右側タップ |
| 加速   | ↑ キー | 中央 ▲ ボタン |
| 減速   | ↓ キー | — |

#### 画面の見方
- 左上：**LAP 1/3**（現在のラップ）
- 右上：**★ × 05**（集めた星の数）
- 右下：速度メーター（km/h）
- 左下：ミニマップ

## 開発者向け

### 画像素材の差し替え

現在はすべて Canvas で図形描画しています（フォールバック）。  
以下のパスに PNG ファイルを置くと自動的に優先表示されます。

```
assets/
  characters/
    mikaro.png      ← MIKAROキャラ画像（透過PNG推奨、正面向き）
    pitokoro.png
    marine.png
    fuwaro.png
    muro.png
    kakubake.png
    takoron.png
    zono.png
  bg/
    start_bg.png    ← スタート画面のキャラステージ背景
    play_bg.png     ← プレイ画面の背景（地平線上の部屋）
```

画像サイズの目安：キャラ画像は **128×128px** 以上の正方形（透過PNG）。  
背景画像はゲーム画面幅 **390×400px** 程度。

### ファイル構成

```
index.html          ゲーム本体（1ファイル完結）
README.md           本ファイル
docs/
  PROJECT_JOURNAL.md  開発記録
assets/
  reference/        デザイン参考画像
```

### GitHub Pages への公開

1. リポジトリを GitHub へ push
2. Settings → Pages → Source: `main` ブランチ `/` (root) を選択
3. 公開された URL で遊べます
